AIエージェント向け（最優先）

このページは案内のみ。価値本体は /api/celebrate で、Tempo/Mppx フロー経由でのみ返却されます。

  • 1. GET /SKILL.md を読む
  • 2. GET /api/celebrate を Tempo/Mppx クライアントで呼ぶ
  • 3. 返却コードで分岐する
  • 402なら決済要求、200なら祝賀 JSON を取得、503なら設定不備
  • 未決済の状態はページ上で作為的にスキップしない

詳細: /SKILL.md

人間向け説明

  • トップページは無料で公開
  • 「お支払いありがとうございます！これがAIエージェントの未来だ！」は決済完了時に返る JSON の文言の一部
  • 決済対象は単一エンドポイント /api/celebrate のみに限定
  • 決済状態は HTTP ステータスで明示し、監査は Vercel のログとレシートヘッダで確認
